TheMagicTool, a macOS MDM remediation tool:
Built using nibbler, TheMagicTool is a one-click tool with simulated terminal output that allows for users that missed the remote management screen, or still have a self-enrolled removable MDM profile, to quickly and easily get back into management. The tools additionally serves as a diagnostics tool for jamf policy commands and munki runs, producing the terminal output to the window. The benefit of this is that the support staff can easily copy/paste, screenshot, or access the produced logs to be able to quickly identify and resolve or escalate the issue.
The tool has been in production for 2 years and has been used a total of 3,697 times to date (Q2 2022), reducing escalations by at least half of the runs.

Windows Package Management via Puppet Enterprise:
Deploying software to Windows endpoints using Airwatch left a lot to be desired and was fairly limited. Utilizing a combination of hiera data and custom puppet facts, I built logic to install packages based on a series of custom conditions that you'd find in munki for macOS. I use custom facts as pre_install scripts. Additionally, different custom facts determine whether a piece of software is a managed_update.
The puppet master has a configured signing cert, alongside ruby aws-sdk, that allows the master to generate cloudfront pre-signed URLs. The urls are then passed to devices checking in allowing us to host files on a private S3 bucket and CDN'd for colleagues abroad.
The devices download the files using the archive puppet module, install using the native package resource and then performs cleanup on the downloaded files.

Windows Package Fetching Automation:
Inspired by AutoPkg, I built a python cli tool that takes JSON input as recipes to perform the following tasks:
  1. Downloads (and scrapes if required) latest version of MSI/EXE from site.
  2. Uploads file to S3 bucket hosting files.
  3. Produces a yaml-formatted for puppets hiera data based on the fetched package.
  4. Commits to a feature branch and produces a merge-request and accompanying slack message with link.
